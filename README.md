# 🎮 C Terminal Game Pack

A simple terminal-based game pack written in C, including:

1. 🔢 Number Guessing Game  
2. 📚 Mini Quiz Game

---

## 💡 Features

- Number guessing with hints (higher/lower)
- Quiz with 3 fun questions
- Input validation for safer interaction
- `switch-case`, `do-while`, `char` input handling

---

## 🚀 How to Play

```bash
gcc game.c -o game
./game
